# Knowledge Proposals

本目录用于存放需要人工审核后才能并入共享真相的 proposal。

适用范围：

- `SOP` 修订提案
- taxonomy 扩展/修订提案
- invariant / fingerprint 演进提案
- 其他由 `report_knowledge_curator` 基于单次 case 提出的知识演进建议

约束：

- proposal 不是 `spec/` SSOT。
- Agent 可以直接在这里新增或更新 proposal，但不得把 proposal 直接合并进 `common/knowledge/spec/`。
- 若 proposal 被正式吸收进 `spec/`，旧 proposal 应在同一轮清理，避免双轨残留。
