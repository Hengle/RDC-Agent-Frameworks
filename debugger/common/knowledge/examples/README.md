# RenderDoc/RDC GPU Debug · Knowledge Examples

本目录存放示例、教学样例、测试夹具与作者参考案例。

允许的内容包括：

- `sessions/`：示例 session artifacts
- `action_chains/`：独立 action chain 示例
- `bugcards/`：BugCard 示例
- `bugfulls/`：BugFull 示例

规则：

- 本目录中的文件不属于 live knowledge 主线
- hooks、validator 与 run compliance audit 不得把本目录中的示例当成当前 run 的真实结案证据
- 真实 session 级 gate artifacts 固定写入 `../library/sessions/<session_id>/`
